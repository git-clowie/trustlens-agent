# TrustLens Visual Assets

This folder contains GitHub and capstone presentation assets.

* `cover/trustlens-cover.png` - primary repository cover image used at the top of `README.md`.
* `gallery/` - compact real-app screenshots used in the README product preview.
* `presentation/` - curated presentation slides copied from the local release handoff.

The local `release/` folder is intentionally ignored by Git, so polished public-facing visuals live here when they should be visible on GitHub.
