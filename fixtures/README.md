# TrustLens Fixtures

Safe synthetic fixtures used for demos, tests, and video recording.

The screenshot assets served by the web demo live in:

* `web/public/fixtures/screenshots/bank-alert.svg`
* `web/public/fixtures/screenshots/courier-fee.svg`
* `web/public/fixtures/screenshots/lottery-claim.svg`
* `web/public/fixtures/screenshots/qr-login.svg`

Each screenshot sample uses a deterministic OCR marker such as `TRUSTLENS_FIXTURE_BANK_SCREENSHOT`, so the local demo behaves consistently with or without Gemini configured.

These fixtures are synthetic and do not represent real messages from the named brands.
